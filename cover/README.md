# Cover
See
[_Draft Documents_](https://sigarra.up.pt/feup/en/WEB_BASE.GERA_PAGINA?P_pagina=258292)
to get the cover template for FEUP  the dissertation cover.

## Setup
### Poppler
```sh
sudo apt install -y poppler-data poppler-utils
```
### PDFtk
```sh
sudo apt install pdftk-java
```

## Usage
### Poppler
```sh
pdfunite cover/cover_two-pages.pdf latex/thesis.pdf thesis.pdf
```
_Note:_ Poppler does not merge the PDF bookmarks into the final file.
### PDFtk
```sh
pdftk cover/cover_two-pages.pdf latex/thesis.pdf output thesis.pdf
```
Use the tool
[https://tools.pdf24.org/en/edit-pdf-bookmarks](https://tools.pdf24.org/en/edit-pdf-bookmarks)
to edit PDF bookmarks automatically generated by
[https://www.ilovepdf.com/organize-pdf](https://www.ilovepdf.com/organize-pdf)
when adding a blank page.
